## Getting Started

### Prerequisites

- python 3.10 or newer
- Tcl/Tk for the GUI.
- Git to download source code.

### Installation

1. Clone the repo
   ```sh
   git clone https://github.com/sauce420/anusooochak.git
   ```
2. Run the `main` file the `gui` folder:
  ```sh
  python3 gui/main.py
  ```
